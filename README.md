📚 Sistema de Biblioteca em Java
📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de Java, como:

Programação Orientada a Objetos (POO)
Encapsulamento
Organização em camadas (model e service)
Manipulação de listas com ArrayList
Tratamento de exceções (try/catch)
Versionamento com Git e GitHub
🚀 Funcionalidades
📖 Cadastro de livros
👤 Cadastro de usuários
🔄 Empréstimo de livros
✅ Devolução de livros
📋 Listagem de livros
🔒 Controle de disponibilidade
