# 📚 Sistema de Biblioteca em Java

## 📌 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Java, incluindo:

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Organização em pacotes (model e service)
- Uso de Collections
- Versionamento com Git e GitHub

---

## 🚀 Funcionalidades (Fase 1)

- Cadastro de livros
- Cadastro de usuários
- Estrutura inicial de uma biblioteca
- Controle básico utilizando listas (ArrayList)

---
